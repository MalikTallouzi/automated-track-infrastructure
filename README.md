## Important Note About Data Files

Due to the large size of the dataset (Data 2), the GPS and vibration files are not included in this repository.

To run the project, you must manually add the required files from Data 2.

Place the files inside:

data2/run1/

Required files:
- GPS.latitude.csv
- GPS.longitude.csv
- GPS.speed.csv
- GPS.satellites.csv
- CH1_ACCEL1Z1.csv
- CH2_ACCEL1Z2.csv

The program uses the GPS data to plot the train route and compare it with the infrastructure data.

Only one dataset (one run) is required to run the code.
